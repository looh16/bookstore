# Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites: 
Any kind of browser. 

### Setup:
    1. Clone the repository
    2. Then open the folder in your local environment 
    3. In terminal console run:
       - npm install
       - npm start
